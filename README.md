# iTan MVP (Iteration 1)

SunAware is a Vue 3 + Django REST Framework prototype that surfaces real-time UV alerts, skin cancer trend visualisations, and contextual protection advice for young adults in Victoria.

## Project layout

```
iTan/
├── backend/              # Django REST API + MySQL-ready schema
├── frontend/             # Vue 3 dashboard UI (Vite)
├── data/                 # Local CSV / seed data used by the API
└── docs/                 # Planning notes & future iteration specs
```

See the detailed instructions in this README and inline comments throughout the codebase.
